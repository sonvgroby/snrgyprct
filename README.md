# Практическая работа

В данном репозитории представлены выполненные задания по практике.

## Структура репозитория

- `case-1-date` — задание 1. Программа на Python для обработки даты рождения.
- `case-2-counter` — задание 2. Веб-приложение «Счётчик» на HTML, CSS и JavaScript.
- `case-3-django` — задание 3. Django-приложение с формой ввода имени и приветствием.
- `case-4-calculator` — задание 4. Веб-приложение «Калькулятор» на HTML, CSS и JavaScript.
- `case-5-analysis` — задание 5. Аналитический обзор программы.
